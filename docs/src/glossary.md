# Glossary

## Other

### `human rights`

- All Mac computer
- A Windows or Linux computer that meets requirements

#### Requirements

- CPU
    - Intel
        - 13th Gen or later
        - Core i5 or more
    - AMD
        - Ryzen 9 3th Gen or later
        - x900X or more
- RAM
    - 16GB or more
- DISK
    - 512GB **SSD** or more
- GPU
    - NVIDIA RTX 20 Series or later
