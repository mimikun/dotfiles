# Product Overview

Cross-platform dotfiles management system for Linux, Windows, and macOS environments.

## Core Capabilities

- **Multi-Platform Configuration**: Unified dotfiles deployment across Linux, Windows (WSL2), and macOS
- **Application Configuration**: Manages configs for shells (fish, bash, zsh), editors (Neovim, Vim), terminals (Alacritty, WezTerm, Foot), and development tools
- **Environment-Specific Logic**: OS-aware and hostname-aware conditional configuration using chezmoi templating
- **Development Workflow Integration**: Pre-configured development environments with Git hooks, linting, formatting, and quality checks

## Target Use Cases

- **Personal Development Environment**: Consistent tooling and settings across multiple machines and operating systems
- **Quick System Bootstrap**: One-command setup for new machines or fresh installations
- **Configuration Synchronization**: Keep development environments in sync across work and personal computers
- **Shell and Editor Customization**: Deeply customized Neovim, Vim, and shell configurations with plugins and enhancements

## Value Proposition

Provides a reproducible, version-controlled development environment that adapts to different platforms while maintaining consistent user experience. Reduces setup time from hours to minutes when migrating to new machines or recovering from system failures.
